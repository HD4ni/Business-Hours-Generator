# 📅 精緻版 IG Story 排班表生成器 (IG Story Schedule Generator)

這是一個專為社群經營者與店家設計的網頁工具，讓你能在幾分鐘內產出具備現代美感、符合 9:16 比例的 Instagram 限時動態排班圖。透過直覺的點擊操作與批次工具，大幅簡化每月營業時間的製作流程。

---
# [點擊這裡進入](https://hd4ni.github.io/IG-Story-Schedule-Generator/)

## ✨ 功能亮點

### 🎨 視覺美學
* **9:16 黃金比例**：專為 Instagram Story 定製，下載後即可直接上傳，無需裁切。
* **現代化排版**：結合 `Montserrat` 現代無襯線字體與 `Playfair Display` 優雅襯線體，營造高級感。
* **毛玻璃效果 (Glassmorphism)**：日期格與說明欄具備半透明與背景模糊效果，能適應各種風格的背景圖。
* **自訂配色**：標題、年份、營業時段與補充說明皆可自由調整顏色。

### ⚡ 效率工具
* **一鍵填滿/取消公休**：快速將全月未設定的日期標記為 `CLOSE`，或一鍵清空所有公休標記。
* **星期批次設定**：點擊週一至週日標題，即可一次性設定全月該星期的營業時段（例如：固定每週三公休）。
* **智慧說明欄**：下方補充說明區塊僅在有文字時顯示半透明底色，保持畫面純淨。

### 🛠️ 操作便利
* **多元背景上傳**：支援直接貼上圖片網址（如 Pinterest, Imgur）或從本地電腦直接上傳圖片。
* **時段精細化**：提供 `08:00` 至 `22:30` 的半小時制選項，每個日期最多可顯示 4 個時段，確保排版不溢出。
* **反應式動畫**：按鈕具備微交互縮放動畫，提供優質的操作回饋。

## 🚀 使用技術

* **HTML5 / CSS3**：使用 Flexbox 與 Grid 佈局，實現精確的介面排列。
* **JavaScript (Vanilla JS)**：純原生 JS 邏輯處理日期生成與批次設定。
* **html2canvas**：將網頁 DOM 元素轉換為高品質 PNG 圖片下載。
* **Google Fonts**：引入高品質網路字體提升質感。

## 📖 操作指南

1.  **設定背景**：
    * 在網址列貼上圖片網址。
    * 或點擊右側的 📁 圖示，從本地裝置上傳背景圖。
2.  **設定月份與顏色**：
    * 確認標題年份是否正確。
    * 使用調色盤調整文字顏色，確保在背景圖上清晰可見。
3.  **安排時段**：
    * **批次設定**：點擊頂部的 `SUN` ~ `SAT` 進行重複性設定。
    * **單日微調**：直接點擊日曆中的日期進行特定調整。
    * **快速公休**：設定完營業日後，點擊「填滿公休」補足剩餘空格。
4.  **補充說明**：
    * 在預覽下方的輸入框填寫備註（如：IG 帳號、預約規則）。
5.  **下載與分享**：
    * 點擊「下載 9:16 日曆圖片」，將生成的圖片存入裝置並上傳至 Instagram。

## 🛠️ 開發說明

若要進行二次開發，請注意以下樣式類別：
* `.day-cell`: 控制日期格的高度與外觀（目前已加長 10%）。
* `.input-group`: 使用絕對定位確保背景圖網址列與上傳按鈕不換行。
* `render()`: 負責核心日曆邏輯，更改 `YEAR` 與 `MONTH` 常數即可切換月份。
